# Учебные материалы к лабораторной работе

Данная директория содержит полный комплект учебных материалов для проведения лабораторной работы по разработке веб-приложений на языке Rust.

## Содержание

1. **lab_work.md** - Текст лабораторной работы с описанием целей, теории и порядка выполнения
2. **group_work_guide.md** - Методические указания по организации групповой работы
3. **library_reference.md** - Справочные материалы по всем используемым библиотекам
4. **resources_collection.md** - Полный список ссылок на документацию и туториалы
5. **git_github_setup.md** - Практикум по SSH‑ключам и подключению локального репозитория к GitHub
6. **git_version_control.md** - Практикум по базовым операциям Git и рекомендованному workflow
7. **beads_guide.md** - Практикум по issue‑трекеру beads и workflow с git

## Для преподавателя

Начните с изучения файла `lab_work.md`, который содержит полное описание лабораторной работы.

## Для студентов

1. Изучите демонстрационный проект в корневой директории
2. Прочитайте текст лабораторной работы (`lab_work.md`)
3. Ознакомьтесь с методическими указаниями по групповой работе (`group_work_guide.md`)
4. Используйте справочник по библиотекам (`library_reference.md`) в процессе работы
5. При работе с GitHub используйте практикум `git_github_setup.md`
6. При работе с Git используйте практикум `git_version_control.md`
7. При работе с задачами используйте практикум `beads_guide.md`
